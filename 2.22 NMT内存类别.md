
本地内存类别：


类别 | 描述
---|---
java heap | 对象的的容器
Class | 类的元信息
Code | 生成的字节码
GC | GC使用的数据接口，例如卡表
Compiler | 编译时生成代码使用的内存
Symbol | 符号
Memory Tracking | NMT自己使用的
Pooled Free Chunks |  Memory used by chunks in the arena chunk pool
Shared space for classes | Memory mapped to class data sharing archive
Thread | 线程使用的数据
Thread Stack | 线程栈
internal |
